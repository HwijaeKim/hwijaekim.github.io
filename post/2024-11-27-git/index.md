## 기본 사용
### 리포지토리 클론
```bash
git clone <리포지토리 URL>
```
   

### 최신 상태로 업데이트
```bash
git pull origin <브렌치 이름>
```
   

### 커밋 및 푸시
```bash
git add .
git commit -m '커밋메시지'
git push
```
